# Evalve Consulting Website

**evalveconsulting.com** — AI Implementation for Education That Actually Works

## About

This is the official website for Evalve Consulting LLC, founded by Dr. Christopher Sanzeri, Ed.D. The site is a single-page design with sections for Home, About, Services, Podcast, Newsletter signup, and Contact.

## Tech Stack

- Pure HTML/CSS/JS (no frameworks, no build step)
- Hosted on GitHub Pages
- Fonts: Instrument Sans (headings) + Nunito Sans (body) via Google Fonts
- Newsletter: Beehiiv (embedded subscribe forms)
- Digital Products: Gumroad (to be added)
- Domain: evalveconsulting.com (registered via Squarespace)

## Editing with Claude Code

You can update this site using Claude Code from your terminal. Examples:

- "Change the hero headline to [new text]"
- "Add a new podcast episode link"
- "Add a Gumroad product card to the services section"
- "Update my email address"
- "Add my photo to the about section"

After making changes, commit and push to deploy:

```bash
git add .
git commit -m "update: description of change"
git push
```

Changes go live automatically via GitHub Pages.
